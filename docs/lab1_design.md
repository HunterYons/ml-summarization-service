# Лабораторная работа №1: Определение задачи и анализ данных

### 1. Предметная область
Автоматизация обработки текстовой информации. Задача: **Abstractive Summarization**.

### 2. Цели и задачи
* **Бизнес-цель:** Экономия времени пользователя при ознакомлении с контентом.
* **Техническая задача:** Построение инференс-пайплайна для модели-трансформера.

### 3. Источник данных
Для оценки и тестов используется датасет **IlyaGusev/gazeta_summarization_top** (корпус новостей Gazeta.ru).

### 4. Выбранные метрики
1. **ROUGE-L:** Оценка качества генерации текста на основе пересечения самых длинных общих подпоследовательностей.
2. **Inference Latency:** Время генерации ответа на один запрос (цель: < 3 сек на CPU).
