# Отчет по Лабораторной работе №3

### 1. Выбранная модель
В качестве baseline использована предобученная модель `mBART` (IlyaGusev/mbart_ru_sum_gazeta).

### 2. Анализ ошибок (Error Analysis)
В ходе тестирования выявлены следующие ограничения:
1. **Галлюцинации:** Иногда модель меняет цифры или путает фамилии, если их много в тексте.
2. **Проблема длинных текстов:** Если статья больше 1024 токенов, модель «забывает» финал статьи (проблема обрезки/truncation).
3. **Стиль:** Модель обучена на новостях, поэтому научные статьи она пытается пересказать в «газетном» стиле.

### 3. Итеративное улучшение
Для улучшения системы планируется добавить очистку текста от специфических символов разметки и ограничение на длину выходной последовательности.
