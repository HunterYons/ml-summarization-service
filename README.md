# ML Summarization Service

Сервис для автоматической суммаризации русскоязычного текста на базе нейросети **RuT5**. Проект реализован в виде микросервисной архитектуры с использованием Docker.

## Технологический стек
* **Python 3.10**
* **FastAPI** — высокопроизводительный API для инференса модели.
* **Aiogram 3.x** — Telegram бот для взаимодействия с пользователем.
* **Transformers (Hugging Face)** — библиотека для работы с моделью `IlyaGusev/rut5_base_sum_gazeta`.
* **Docker & Docker Compose** — контейнеризация и оркестрация сервисов.

## Архитектура проекта
Проект состоит из двух основных контейнеров:
1.  **api**: Бэкенд на FastAPI. Загружает веса модели и предоставляет эндпоинт `/summarize`.
2.  **bot**: Telegram бот, который принимает текст от пользователя и пересылает его в контейнер API по внутренней сети Docker.

## Быстрый запуск

### 1. Подготовка
Создайте файл `.env` в корневой директории и добавьте туда ваш токен:
```env
TELEGRAM_TOKEN=ваш_токен_бота
API_URL=http://api:8000/summarize
2. Запуск через Docker
Убедитесь, что у вас запущен Docker Desktop, и выполните команду в терминале:
docker-compose up --build

После завершения сборки бот будет готов к работе.

Использование
Найдите бота в Telegram и нажмите /start.

Отправьте любой длинный текст на русском языке.

Бот вернет краткую выжимку (суммаризацию), сгенерированную нейросетью.

Логирование
Запросы к API и результаты работы модели логируются внутри контейнера в файл api_history.log.
