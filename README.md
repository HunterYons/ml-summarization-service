# Сервис суммаризации веб-страниц (ML System Design)

Проект по курсу «Проектирование интеллектуальных информационных систем» (ПИИС).

## Обзор проекта
Цель проекта — разработка работающего прототипа ML-сервиса для автоматической генерации краткого содержания (суммаризации) новостных текстов. Сервис принимает на вход текст или URL и возвращает сжатую выжимку основных мыслей.

## Структура репозитория
* `docs/` — документация по системному дизайну (ЛР №1, №2).
* `src/` — исходный код сервиса (ЛР №3, №4).
* `requirements.txt` — список зависимостей для развертывания.

## Быстрый запуск
1. Установка зависимостей: `pip install -r requirements.txt`
2. Запуск baseline-модели: `python src/model.py`
